TLEX

Lexer Generator for TypeScript

Build powerful tokenizers and lexical analyzers with an intuitive API. Supports JavaScript regex and Flex syntax.

Get Started View on GitHub

Multiple Regex Syntaxes

Use JavaScript RegExp or Flex-style extended syntax. Choose what works best for your use case.

High Performance

VM-based regex matching with efficient bytecode compilation for fast tokenization.

Flexible Rules

Define rules with priorities, states, and callbacks for complete control over tokenization.

TypeScript First

Written in TypeScript with full type definitions. Works seamlessly in modern JavaScript projects.

Quick Example

import { Tokenizer } from 'tlex';

const lexer = new Tokenizer();
lexer.add(/[0-9]+/, { tag: "NUMBER" });
lexer.add(/[a-z]+/, { tag: "IDENT" });
lexer.add(/\s+/, { skip: true });

const tokens = lexer.tokenize("abc 123 def");
// [Token(tag="IDENT"), Token(tag="NUMBER"), Token(tag="IDENT")]
